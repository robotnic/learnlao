Drop MP3 files here.

File naming convention
- Put audio files at: `apps/learnlao/src/assets/audio/<audio_key>.mp3`
- The Dictionary page will request: `/assets/audio/<audio_key>.mp3`

Example
- If a word has `"audio_key": "sabaidee_v1"`, create:
  - `apps/learnlao/src/assets/audio/sabaidee_v1.mp3`
